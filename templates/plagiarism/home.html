What is GBPUAT SIMILARITY CHECKER

Gbpuat Text Similarity Checker system is a web app that uses text similarity algorithms, OCR and web scrapping in order to check plagiarism in assignments submitted by students. The students have the option to submit assignments in digital or handwritten image format. The web app compares the assignments to assignments of other students as well as to web resources like Geeks For Geeks


Our Features

CLEAN CODES

Easy to understand and open source code that is well documented that makes it easier to customize and improve the web app.

Extensive Database

The app has a database of about 8k articles published on Geeks for Geeks

Human Friendly Result

The app uses human friendly algorithm like Sequence Matcher in the Front end to display plagiarised texts while uses more complex BERT algo to display plagiarism score

Syntax and Semantic Similarity

Handles both types of similarities- Syntax similarity(same words used) as well as semantic similarity (re-phrasing of sentences)
